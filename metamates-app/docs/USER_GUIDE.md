# MetaMates 用户指南

> 最后更新：2026-07-10

---

## 产品简介

**MetaMates = 私人灵感仓库 + 思考引擎。** 本地 Markdown 仓库承载碎片与笔记，**右侧思考引擎是日常起点**（对话与 15 条 slash 命令），**中间编辑器用于查看与修改写回的成品**。输出落盘到本地文件，而不是消失在聊天窗口里。

---

## 快速开始

### 安装

1. 打开 [GitHub Releases](https://github.com/qdljywz/MetaMates/releases)，下载适合你系统的安装包（若暂无附件，请用源码构建便携版，见仓库根 README）
2. 安装或解压后运行 **MetaMates**

### 首次使用

1. 首启向导：选择灵感仓库文件夹（自动初始化目录结构）
2. 安装并连接至少一个 AI 助手（Gemini / Claude / CodeBuddy）
3. 在右侧思考引擎发消息或跑 `/today`；成品在中间编辑器打开、修改

## 界面介绍

### 布局：思考引擎为主入口

```
┌──────────┬─────────────────────────┬──────────────────┐
│ 活动栏    │  编辑器（多标签）         │  思考引擎         │
│ 文件树    │  Markdown · 双链 · 标签  │  AI 对话 + slash │
│ 日记日历  │  自动保存                │  命令 + 工具卡片  │
│ 搜索/图谱 │                          │                  │
└──────────┴─────────────────────────┴──────────────────┘
```

### 活动栏（左侧）

| 按钮 | 功能 |
|------|------|
| 📁 文件树 | 浏览和管理文件（需先打开工作区） |
| 📅 日记日历 | 点击日期创建/打开日记；Shift+点击打开 PLAN |
| 🔍 搜索 | 全局搜索（Ctrl+Shift+F） |
| 📊 关系图谱 | 可视化文件关联 |
| 📝 新建笔记 | 在 Inbox 创建新文件（需已打开工作区） |
| 📂 打开工作区 | 选择或切换工作区文件夹 |

### 思考引擎（右侧）

| 区域 | 功能 |
|------|------|
| 工具栏 | 切换 AI 助手（Gemini / Claude / CodeBuddy）、连接状态、模型与模式 |
| 消息区 | 对话与工具调用卡片；空历史时显示灵感仓库上下文 |
| 底部 | 横向 slash 命令 chips + 输入框（支持 `@` 引用文件、拖拽附件） |

**原则**：每位 AI 助手 **一条持续对话**。思考引擎完成读写后，可在中间编辑器查看、微调目标文件。

## 核心功能

### 1. 文件管理

#### 文件树操作

| 操作 | 说明 |

| 右键菜单 | 新建、重命名、删除等 |

> 未打开工作区时，文件树空状态提供「打开工作区」按钮；不会在没有工作区时新建笔记。

#### 右键菜单

**文件夹菜单**：
- 展开/折叠
- 新建笔记
- 新建子文件夹
- 使用模板
- 重命名
- 删除
- 复制路径
- 在资源管理器中打开

**文件菜单**：
- 重命名
- 删除
- 复制路径
- 在资源管理器中打开
- 导入为情报笔记（支持的格式）

#### 拖拽移动

- 拖拽文件到文件夹 → 移动文件
- 拖拽文件夹到文件夹 → 移动文件夹

### 2. Markdown 编辑器

#### 基本功能

| 功能 | 说明 |
|------|------|
| 语法高亮 | 支持 Markdown 语法 |
| 自动保存 | 内容变化时自动保存（可在设置中关闭） |
| 多标签页 | 同时编辑多个文件 |
| 无标签占位 | 关闭全部标签后顶栏保留细条，显示「思考引擎优先」空态（个性化问题 + 下一步建议），引导你直接与右侧 AI 交互 |

#### 空态（思考引擎优先）

当你关闭全部标签后，编辑器会显示一条**随时间与工作记录变化**的问题（默认按你设置的时区计算，例如北京时间 `Asia/Shanghai`），并提供：

- **和 AI 聊这个**：自动把问题填入右侧输入框并聚焦
- **换一个问题**：手动刷新一次问题
- 若你有 Inbox 待整理、今日计划未生成、最近编辑过的文件等，会在下方给出快捷动作

> 空态问题会在后台每 10 分钟静默更新一次（并本地缓存），所以下次回到空态时通常会看到新的建议。

#### Wiki 链接

使用 `[[文件名]]` 语法创建链接：

```markdown
查看 [[2026-03-20 PLAN]] 了解今日计划
```

点击链接可跳转到目标文件。

#### 标签

使用 `#标签` 语法添加标签：

```markdown
这是一个重要的想法 #重要 #待办
```

点击标签可搜索包含该标签的所有文件。

### 3. 思考引擎与 slash 命令

#### 使用方式

1. 在右侧**思考引擎**顶部选择 AI 助手（需本机已安装并登录）
2. 点击底部 slash chip（如 `/today`）或手动输入命令
3. AI 助手通过工具读写灵感仓库中的 Markdown；完成后可在编辑器中查看结果

#### 命令列表（15 条）

| 命令 | 功能 |
|------|------|
| `/context` | 加载项目上下文 |
| `/today` | 生成今日计划 |
| `/closeday` | 每日复盘 |
| `/schedule` | 时间块分配 |
| `/trace` | 溯源想法 |
| `/connect` | 发现笔记关联 |
| `/challenge` | 挑战观点 |
| `/ghost` | 模拟代写 |
| `/ideas` | 生成点子报告 |
| `/graduate` | 灵感升级 |
| `/drift` | 潜意识漂移 |
| `/emerge` | 项目涌现 |
| `/intel` | 情报导入（网页/文档 → `04_情报与连接/`） |
| `/sync` | 同步全库进展 |
| `/soal` | 进化固化（写入 `2M.md`） |

#### 情报导入（四种入口）

| 入口 | 说明 |
|------|------|
| **`/intel`（思考引擎）** | 粘贴链接或文件路径，本地抓取 + AI 助手深化 |
| **直接发链接** | 在思考引擎输入框发送 URL，自动识别入库 |
| 文件树右键 | 对 PDF/Word/表格/图片等 →「导入为情报笔记」 |
| 命令面板 `Ctrl+P` | 搜索「导入外部文档为情报」 |

导入后建议：**核对笔记内容** → **添加 `[[双链]]` 与 `#标签`** 连接到现有项目。

完整手册：应用内点击标题栏 <strong>?</strong> →「打开完整使用手册」；或在本仓库打开 [user-manual.html](user-manual.html)。

#### 快捷提示

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+P | 打开/创建今日 PLAN |

> `/soal` 等 slash 命令请在右侧思考引擎输入，无全局快捷键。完整列表见应用内帮助或 [user-manual.html](user-manual.html) §07。

## 常见问题

### Q: 如何更改主题？

A: 点击右上角的设置按钮，可以切换深色/浅色主题。

### Q: 数据存储在哪里？

A: 所有数据存储在你选择的工作区目录中，以 Markdown 格式保存。

### Q: 如何备份数据？

A: 直接复制工作区目录即可完成备份。

### Q: 如何配置 AI 服务？

A: 点击右上角 **设置 → Agent**，点击 **安装 AI 助手** 安装 Gemini、Claude 或 CodeBuddy 等助手，并按提示完成首次登录。MetaMates 通过本机 ACP CLI 连接助手，无需在应用内单独填写 API Key（具体认证方式因助手而异）。完整步骤见应用内 **?** →「打开完整使用手册」。

### Q: 代理/系统时区不准，计划与日记日期会错怎么办？

A: 在右上角 **设置 → 常用** 中找到「**时区**」，填写你的 IANA 时区（例如北京时间填写 `Asia/Shanghai`）。  
之后：

- 日历、`/today`、PLAN/日记文件名的日期计算会按该时区执行
- AI 助手 prompt 会注入「生效时区 + 当前本地时间」，避免 CLI 误判日期

### Q: 终端中文显示乱码怎么办？

A: 终端已支持 UTF-8 编码，如果仍有问题，请检查系统编码设置。
