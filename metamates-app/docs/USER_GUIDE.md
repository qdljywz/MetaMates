# Metamates 用户指南

> 最后更新：2026-06-22

---

## 产品简介

Metamates 是**本地优先的个人知识操作系统**：Markdown Vault 承载思考与规划，右侧 Agent 执行层通过本机 CLI（ACP）在同一工作区里读写笔记。日常可替代 Obsidian 记笔记，并多一层「可行动的 AI」——输出写回 Markdown，而不是消失在聊天窗口里。

---

## 快速开始

### 安装

1. 下载 `Metamates.exe`
2. 双击运行即可

### 首次使用

1. 首启向导：选择工作区文件夹（自动初始化目录结构）
2. 检测并连接至少一个 CLI Agent（Gemini / Claude / CodeBuddy）
3. 在中间编辑器写笔记，在右侧 Agent 面板运行 `/today` 等命令

## 界面介绍

### 布局：Vault 主场 + Agent 执行层

Metamates **不是**独立 Chat App。编辑器是主轴，Agent 是右侧窄栏执行层。

```
┌──────────┬─────────────────────────┬──────────────────┐
│ 活动栏    │  编辑器（多标签）         │  Agent 执行层     │
│ 文件树    │  Markdown · 双链 · 标签  │  CLI 图标 + 状态  │
│ 日记日历  │  自动保存                │  slash chips     │
│ 搜索/图谱 │                          │  输入 + 工具卡片  │
└──────────┴─────────────────────────┴──────────────────┘
```

### 活动栏（左侧）

| 按钮 | 功能 |
|------|------|
| 📁 文件树 | 浏览和管理文件 |
| 📅 日记日历 | 点击日期创建/打开日记；Shift+点击打开 PLAN |
| 🔍 搜索 | 全局搜索（Ctrl+Shift+F） |
| 📊 关系图谱 | 可视化文件关联 |
| 📝 新建笔记 | 创建新文件（写入 Inbox） |
| 📂 打开工作区 | 切换工作区 |

### Agent 面板（右侧）

| 区域 | 功能 |
|------|------|
| 工具栏 | 切换 CLI（Gemini / Claude / CodeBuddy）、连接状态、模型与模式 |
| 消息区 | 对话与工具调用卡片；空历史时显示 Vault 工作区上下文 |
| 底部 | 横向 slash 命令 chips + 输入框（支持 `@` 引用文件、拖拽附件） |

**原则**：每个 CLI **一条持续对话**，不做多会话/对话树。Agent 完成读写后，可自动或手动跳转到编辑器中的目标文件。

## 核心功能

### 1. 文件管理

#### 文件树操作

| 操作 | 说明 |
|------|------|
| 单击文件夹 | 展开/折叠文件夹 |
| 单击文件 | 打开文件 |
| 双击文件 | 在新标签页打开 |
| 右键菜单 | 更多操作 |

#### 右键菜单

**文件夹菜单**：
- 展开/折叠
- 新建笔记
- 新建子文件夹
- 使用模板
- 重命名
- 删除
- 复制路径
- 在资源管理器中打开

**文件菜单**：
- 打开
- 在新标签页打开
- 重命名
- 删除
- 复制路径
- 在资源管理器中打开

#### 拖拽移动

- 拖拽文件到文件夹 → 移动文件
- 拖拽文件夹到文件夹 → 移动文件夹

### 2. Markdown 编辑器

#### 基本功能

| 功能 | 说明 |
|------|------|
| 语法高亮 | 支持 Markdown 语法 |
| 自动保存 | 内容变化时自动保存 |
| 多标签页 | 同时编辑多个文件 |
| 标签页拖拽 | 调整标签页顺序 |

#### Wiki 链接

使用 `[[文件名]]` 语法创建链接：

```markdown
查看 [[2026-03-20 PLAN]] 了解今日计划
```

点击链接可跳转到目标文件。

#### 标签

使用 `#标签` 语法添加标签：

```markdown
这是一个重要的想法 #重要 #待办
```

点击标签可搜索包含该标签的所有文件。

### 3. Agent 执行层与 slash 命令

#### 使用方式

1. 在右侧 Agent 面板顶部选择 CLI（需本机已安装并登录）
2. 点击底部 slash chip（如 `/today`）或手动输入命令
3. Agent 通过工具读写 Vault 中的 Markdown；完成后可在编辑器中查看结果

#### 命令列表（15 条）

| 命令 | 功能 |
|------|------|
| `/context` | 加载项目上下文 |
| `/today` | 生成今日计划 |
| `/closeday` | 每日复盘 |
| `/schedule` | 时间块分配 |
| `/trace` | 溯源想法 |
| `/connect` | 发现笔记关联 |
| `/challenge` | 挑战观点 |
| `/ghost` | 模拟代写 |
| `/ideas` | 生成点子报告 |
| `/graduate` | 灵感升级 |
| `/drift` | 潜意识漂移 |
| `/emerge` | 项目涌现 |
| `/intel` | 情报导入（网页/文档 → `04_情报与连接/`） |
| `/sync` | 同步全库进展 |
| `/soal` | 进化固化（写入 `2M.md`） |

#### 情报导入（四种入口）

| 入口 | 说明 |
|------|------|
| Agent `/intel` | 粘贴链接或文件路径，本地抓取 + Agent 深化 |
| 直接发链接 | 在 Agent 输入框发送 URL，自动识别入库 |
| 文件树右键 | 对 PDF/Word/表格/图片等 →「导入为情报笔记」 |
| 命令面板 `Ctrl+P` | 搜索「导入外部文档为情报」 |

导入后建议：**核对笔记内容** → **添加 `[[双链]]` 与 `#标签`** 连接到现有项目。

完整手册见 [user-manual.html](user-manual.html)。

#### 快捷键

| 快捷键 | 功能 |
|--------|------|
| Ctrl+Shift+P | 打开/创建今日 PLAN |
| Ctrl+Shift+O | 运行 `/soal` |

## 工作区结构

Metamates 会自动初始化以下文件夹结构：

```

├── 01_日记与计划_Daily_Log/     # 每日笔记和计划
├── 02_项目与知识_Knowledge_Base/ # 项目文档和知识库
├── 03_点滴积累_Insights/         # 灵感和想法
```

## 常见问题

### Q: 如何更改主题？

A: 点击右上角的设置按钮，可以切换深色/浅色主题。

### Q: 数据存储在哪里？

A: 所有数据存储在你选择的工作区目录中，以 Markdown 格式保存。

### Q: 如何备份数据？

A: 直接复制工作区目录即可完成备份。

### Q: 如何配置 AI 服务？

A: 点击右上角的设置按钮，在设置面板中配置 API Key。

### Q: 终端中文显示乱码怎么办？

A: 终端已支持 UTF-8 编码，如果仍有问题，请检查系统编码设置。
