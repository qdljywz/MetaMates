## 🔄 版本历史

### v0.2-beta (2026-03-25)

**新增功能**：
- 国际化支持 (i18n) - 完整的中英文切换
- 主题系统完善 - 深色/浅色主题完整支持
- 终端 AI 命令快捷按钮 - 终端面板中显示 13 个 AI 命令
- 命令智能分类 - 区分直接执行和需要输入的命令
- 命令输入弹窗 - 需要输入的命令弹出输入框

### v0.1-beta (2026-03-20)

- 关系图谱可视化
- 终端集成（xterm.js + node-pty）
- 文件树单击展开/折叠
