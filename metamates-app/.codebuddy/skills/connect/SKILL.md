# 🔗 寻找连接 (Connect)

请执行以下任务：

1. **接收两个主题**：用户提供主题 A 和主题 B
2. **分别搜索**：在库中搜索两个主题的相关笔记
3. **寻找交集**：识别连接两个主题的笔记或概念
4. **分析模式**：展示发现的联系和潜在模式

## 输出格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔗 主题连接分析
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📌 主题 A：[名称]
相关笔记：
- [[笔记1]] - [摘要]
- [[笔记2]] - [摘要]

📌 主题 B：[名称]
相关笔记：
- [[笔记3]] - [摘要]
- [[笔记4]] - [摘要]

🔗 发现的连接
- [[桥梁笔记]]：[如何连接两者]

💡 潜在模式
- [模式1]：[说明]
- [模式2]：[说明]

🎯 启发
- [启发1]
- [启发2]
```
