# 🎓 灵感升级 (Graduate)

请执行以下任务：

1. **扫描日记**：读取 `01_日记与计划/` 最近 14 天的记录
2. **识别可升级灵感**：发现值得独立成文的碎片化想法
3. **创建永久笔记**：在 `03_点滴积累/` 创建独立的 Zettel 笔记
4. **回链**：在原日记中添加指向新笔记的链接

## 输出格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🎓 灵感升级报告
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 扫描范围：最近 14 天

📋 发现的可升级灵感

1. [[新笔记1]] (来源：[[日记A]])
   > [灵感摘要]

2. [[新笔记2]] (来源：[[日记B]])
   > [灵感摘要]

✅ 已创建的永久笔记
- `03_点滴积累/[笔记名].md`

🔗 已更新的原日记
- [[日记A]] - 添加了 → [[新笔记1]]
- [[日记B]] - 添加了 → [[新笔记2]]

💡 升级原则
- 主题独立，可脱离原日记理解
- 有进一步探索的价值
- 与其他笔记有潜在连接
```
