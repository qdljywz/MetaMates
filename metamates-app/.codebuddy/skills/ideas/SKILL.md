# 💡 点子报告 (Ideas)

请执行以下任务：

1. **扫描全库**：搜索 `01_日记与计划/`, `02_项目与知识/`, `03_点滴积累/`
2. **识别新兴模式**：发现反复出现但尚未系统化的主题
3. **生成点子报告**：分类为工具、人脉、写作主题、探索方向

## 输出格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 扫描范围：最近 [N] 天

🛠️ 应该构建的工具
1. [工具1] - 来源：[[笔记X]]
2. [工具2] - 来源：[[笔记Y]]

👥 应该联系的人
1. [人名] - 原因：[说明]
2. [人名] - 原因：[说明]

✍️ 应该写作的主题
1. [主题1] - 素材来源：[[笔记A]], [[笔记B]]
2. [主题2] - 素材来源：[[笔记C]]

🔍 应该探索的方向
1. [方向1] - 为什么：[说明]
2. [方向2] - 为什么：[说明]

📊 新兴模式摘要
- [模式1]：出现了 [N] 次
- [模式2]：出现了 [N] 次
```
