# 🔍 溯源想法 (Trace)

请执行以下任务：

1. **接收主题输入**：用户指定要追踪的主题或关键词
2. **全库搜索**：在所有笔记中搜索该主题的所有提及
3. **跟踪回链**：查找相关的反向链接和引用
4. **生成时间线**：按时间顺序展示想法的演变过程

## 输出格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔍 想法溯源：[主题]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 演变时间线

[日期1] - [文件1]
> [相关内容摘要]

[日期2] - [文件2]
> [相关内容摘要]

[日期3] - [文件3]
> [相关内容摘要]

📊 演变分析
- 起源：[最初的想法]
- 演变：[中间的变化]
- 现状：[当前的理解]

🔗 相关链接
- [[文件1]]
- [[文件2]]
```
