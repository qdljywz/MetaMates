# ⚔️ 挑战观点 (Challenge)

请执行以下任务：

1. **接收主题**：用户提供要挑战的观点或主题
2. **搜索相关笔记**：找出所有关于该主题的记录
3. **识别矛盾**：寻找自我矛盾或不一致的陈述
4. **质疑假设**：挖掘潜在的隐含假设并质疑其有效性

## 输出格式
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
⚔️ 观点挑战：[主题]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📝 你的观点摘要
- [观点1]
- [观点2]

⚠️ 发现的矛盾
- [[笔记A]] 说："..."
- [[笔记B]] 说："..."
→ 矛盾点：[说明]

🤔 潜在的错误假设
1. [假设1] - 为什么可能错误
2. [假设2] - 为什么可能错误

💡 挑战问题
- 如果 [条件] 不成立，会怎样？
- 有什么证据反驳这个观点？

🔄 建议修正
- [修正建议1]
- [修正建议2]
```
