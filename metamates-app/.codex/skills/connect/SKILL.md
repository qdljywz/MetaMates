# 🔗 寻找连接 (Connect)

请执行以下任务：

1. **接收两个主题**：用户提供主题 A 和主题 B
2. **分别搜索**：在库中搜索两个主题的相关笔记
3. **寻找交集**：识别连接两个主题的笔记或概念
4. **分析模式**：展示发现的联系和潜在模式
