# 情报与连接

用于存储外部情报、人脉连接、行业动态等信息。

- **记忆索引**：[[记忆索引]] — 用户可读的长期记忆（Agent 须同步摘要到此）
- **参考条目**：`参考/` — 合作方、外部资源等详细条目

## 使用建议

- 保存行业报告和研究资料
- 记录重要人脉信息
- 收集竞争对手情报
- 整理市场动态和趋势

## 导入方式

| 方式 | 说明 |
|------|------|
| **文件树右键** | 对 PDF / Word / Excel / CSV / 图片等 →「导入为情报笔记」 |
| **命令面板** `Ctrl+P` | 搜索「导入外部文档为情报」 |
| **Agent `/intel`** | 粘贴网页链接或文件路径，自动抓取并写入本目录 |
| **直接发链接** | 在 Agent 对话框发送 URL（或「总结 https://…」），同样会入库 |

原件归档在 `sources/` 子目录；摘要笔记为同目录下的 `.md` 文件，会进入知识索引与图谱。
