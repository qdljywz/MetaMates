# 📡 情报导入 (/intel)

## 流程说明

Metamates 桌面端会先完成**本地抓取**（网页、PDF、Word、Excel、图片 OCR 等），在 `04_情报与连接/` 生成草稿笔记，原件归档于 `sources/`。你的任务是**深化**该笔记：

1. **阅读**刚创建的情报 `.md` 及来源信息（URL 或 `sources/` 路径）
2. **补充**结构化摘要：核心观点、关键数据、待验证假设
3. **连接**：添加 `#标签` 与 `[[双链]]` 到库内相关项目/笔记
4. **行动**：在「行动建议」区块列出可跟进事项

## 用户输入

- 网页 URL（http/https）
- 或工作区内文档相对路径（PDF / 图片 / docx / xlsx / csv / md 等）

## 写回要求

- 目标目录：`04_情报与连接/`（草稿 `.md` 已存在则**更新**而非重复创建）
- 保留 frontmatter 中的来源、导入时间等元数据
- 写回后读取文件确认更新成功
