# 🧠 Skill: /soal (Evolutionary Sync)

## 📌 Description
将用户的反馈、习惯或新教训永久固化到 Metamates 的核心进化层 `2M.md` 中。

## 🛠️ Usage
输入：`/soal [您的习惯或教训]`

## 📜 Instructions
当收到 `/soal` 指令时，请执行以下操作：
1. **解析输入**：提取用户提供的核心观点或行为模式。
2. **读取大脑**：读取 `05_模板与配置/2M.md`。
3. **分类更新**：
   - 如果是个人偏好（如语言、语气），更新至 `## 🧬 I. 用户 DNA`。
   - 如果是技术习惯（如代码审美、命名规则），更新至 `## 🛠️ II. 战术协议`。
   - 如果是纠错（如“不应该用某库”），更新至 `## ⚠️ III. 错误记忆与修正`。
4. **记录日志**：在 `## 📈 IV. 主动学习日志` 中追加当前日期和更新条目。
5. **反馈确认**：以简明扼要的方式告知用户该进化点已永久固化，并展示受影响的代码审美或决策逻辑。
