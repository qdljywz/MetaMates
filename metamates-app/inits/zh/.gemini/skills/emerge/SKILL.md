# 🚀 项目涌现 (Emerge)

请执行以下任务：

1. **全库扫描**：搜索所有笔记和相关内容
2. **聚类分析**：识别主题相关、可以聚合的笔记群
3. **涌现识别**：发现正在形成中的项目、文章或产品雏形
4. **涌现报告**：列出潜在项目及其组成要素
