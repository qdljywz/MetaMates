# 🧠 Skill: /soal (Evolutionary Sync)

## 📌 Description
Permanently solidify user feedback, habits, or new lessons into Metamates' core evolutionary layer, `2M.md`.

## 🛠️ Usage
Input: `/soal [Your habit or lesson]`

## 📜 Instructions
When receiving the `/soal` command, please perform the following:
1. **Parse Input**: Extract the core viewpoint or behavior pattern provided by the user.
2. **Read Mind**: Read `05_Templates_and_Config/2M.md`.
3. **Categorize and Update**:
   - If it's a personal preference (e.g., language, tone), update `## 🧬 I. User DNA`.
   - If it's a technical habit (e.g., code aesthetics, naming rules), update `## 🛠️ II. Tactical Protocols`.
   - If it's a correction (e.g., "should not use a certain library"), update `## ⚠️ III. Error Memory & Correction`.
4. **Log Journal**: Append the current date and update entry to `## 📈 IV. Active Learning Log`.
5. **Confirm Feedback**: Inform the user in a concise manner that the evolution point has been permanently solidified, and show the affected code aesthetics or decision logic.
