---
name: trace
description: 溯源想法 - 追踪某个特定想法在整个库中是如何演变的轨迹
---

请执行以下任务：

1. **接收主题**：用户输入一个主题或关键词
2. **全库搜索**：在Metamates库中搜索该主题的所有提及
3. **追踪时间线**：按时间顺序整理该想法的演变过程
4. **分析演变**：识别想法如何从一个笔记发展到另一个笔记
5. **输出轨迹图**

输出格式：
```
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
🔍 想法溯源：[主题]
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

📅 演变时间线
[日期] - [文件名]
├─ 内容摘要：[该笔记中关于此主题的核心内容]
└─ 连接到：[下一个相关笔记]

[日期] - [文件名]
├─ 内容摘要：[...]
└─ 连接到：[...]

- 相关笔记总数：[X]

- [关于这个想法演变的观察]
```
